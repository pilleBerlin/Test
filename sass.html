SASS

Sass Dateien sind eine Erweiterung von CSS und werden im Format .scss gespeichert

Innerhalb der CSS Dateien können jetzt auch Variablen gespeichert werden. Bsp. $color: red; ul{ font-size: 12 px; color: $color; }...
Da jedoch die Browser nicht alle .scss verstehen müssen die Datei(en) konvertiert werden.

Das passiert mit dem Befehl: sass "dateiname.scss" "neuedateiname.csss"

Dadurch wird die Datei "neuedatename.css" generiert, eisnchließlich einer .css.map Datei für's Debugging zB

Ich kann auch SASS dazu bringen die Datei automatisch zu kompilieren

Befehl hierzu: sass --watch dateiname.scss:neuedateiname.css

Dadurch wird diese Datei beobachtet und sobald eine Veränderung erfolgt, wird diese Datei gemäß Vorgabe verändert
