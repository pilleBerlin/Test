id in CSS ist eindeutig > #idbezeichnung | # | einmal verwenden

class sollte mehrfasch verwendet werden > .classbezeichnung | . | mehrfachverwendung möglich

Bei verschachtelten Elementen wie bei einer Liste, wenn da nur ein Knotenpunkt referenziert werden soll, da schreibt man "ol > li". D.h. nur das Element li unter dem Elemenent ol wird via CSS berührt
