git Befehle

  • git add //Diese Datei wird überwacht und dem Repository zur Verfügung gestellt
  • git commit -m "message" //Mach einen Snapshot und sichere es im lokalen Repository
  • git status //gibt den aktuellen Status zurück, bsp. was zu tun ist, welcher branch etc.
  • git push // alle Daten die markiert sind und commitet, werden an git gesendet
  • git pull //alle neuen Daten werden heruntergeladen
